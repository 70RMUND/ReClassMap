ReClassMap Documentation

 

Class: Map(…)

Description:

                This is the main ReClassMap object that compiles all the python node descriptions and translates them into *.reclass XML schema. Invoking the write() function will cause the Map engine to elaborate all assigned node classes and their children nodes

Constructor Arguments:

·         file = [string, filepath to write *.reclass file]

Object Functions:

·         add_class(cls)

o   This function adds a Class() object into the class directory for this Map() object

o   No return value

·         Class(<Class() Constructor Arguments>)

o   This function creates a Class() object and associates it to this Map() object

o   Return: the Class() object created

o   Note: this is just a faster way to create and associate a Class() object

·         write(hidepad=0,custompad=0)

o   This function causes the Map engine to elaborate all assigned node classes and their children nodes. The result is a *.reclass 2016 XML payload that is saved to the path specified by file

o   No return value

o   Argument hidepad accepts int 0 or 1 (default 0). This tells the elaboration engine to mark all generated pad as hidden from the user

o   Argument custompad accepts int 0 or 1 (default 0). This tells the elaboration engine to use Custom() node objects instead of Hex*() node object to model as padding

Class: Class(…)

Description:

                This ReClassMap object models a Class nodetype in ReClass. An instance can be generated by the Map() object for a 1-line create and associate. The Class object is a special node type in charge of keeping track of all children node objects. Class objects allow for assignment into its memory map using indexing and will autopad undefined memory.

Constructor Arguments:

·         name = [string, the name of this node]

·         offset = [int, optional, default=0, specify a static offset for this class]

·         size = [int, optional, default=0, specify a size for this class node to pad to]

·         comment = [string, optional, default=””, specify an inline ReClass comment for this node]

·         empty = [bool, optional, default=False, used internally to mark this class node as an auto generated node (do not use)]

Object Functions:

·         The most used functions of this object is the index assignment overload. All other node class object get associated in the Class memory map by leveraging these overloaded functions

Class: Vtable(…)

Description:

                This node type object creates and models an nt_vtable ReClass node. This object has no public functions and is meant to be assigned into a memory index of a Class() object. This object keeps a directory of virtual function objects. Like the Class() object you can assign a Vfunc() object to it by indexing into the object at a specific virtual function number. Virtual functions which are not defined will be auto generated by the elaboration engine

Constructor Arguments:

·         name = [string, the name of this node]

·         comment = [string, optional, default=””, specify an inline ReClass comment for this node]

Class: Vfunc(…)

Description:

                This object is not a node object but a virtual function object declaration. This object has no public functions and is meant to be assigned into a virtual function index of a Vtable() object

Constructor Arguments:

·         name = [string, the name of this virtual function]

·         comment = [string, optional, default=””, specify an inline ReClass comment for this virtual function]
